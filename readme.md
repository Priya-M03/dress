# Dress Website Project

A simple responsive dress shopping website built using HTML, CSS, and JavaScript.

## Project Overview
This project is a frontend web application designed to display dress collections in an attractive user interface. It demonstrates the use of core web technologies for creating interactive and responsive webpages.

## Technologies Used
- HTML5
- CSS3
- JavaScript

dress/

## Features
- Responsive webpage design
- Attractive user interface
- Interactive JavaScript functionality
- Clean and simple layout
